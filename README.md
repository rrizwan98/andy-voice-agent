# Andy — AI Voice Employee

Talk to Andy on a phone call. Enter your number, click "Call Me", and Andy calls you back.

**Andy can build:** websites, AI agents, APIs, full-stack projects — from a single WhatsApp message, 24/7.

## Environment Variables Required

Set these in your Space settings:

- `OPENAI_API_KEY`
- `OPENAI_WEBHOOK_SECRET`
- `OPENAI_PROJECT_ID`
- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
- `TWILIO_PHONE_NUMBER`
- `BASE_URL` → `https://rrizwan98-openai-first-voice-agent.hf.space`
